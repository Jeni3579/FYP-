How to Use the IT Security Dashboard

This guide explains the features of your real-time threat analysis tool.

1. Live Threat Analysis Chart

The main bar chart on your dashboard displays a live count of all detected events, separated into four categories:

  • Normal: Regular, safe system activity.
  • Potential Malware: Events that match patterns of malware behavior.
  • Unauthorized Access: Suspicious login attempts.
  • Ransomware Activity: Events that indicate a ransomware attack.

2. Live Security Alerts Feed

The feed on the right side of the dashboard shows individual threat alerts as they happen. Each alert is color-coded by its severity:

  • Yellow/Orange alerts indicate low-to-medium risk events.
  • Red alerts indicate high-risk events like ransomware activity.

3. Drill-Down for Details (Interactive)

For more information on any threat, simply **click on an alert** in the live feed. A pop-up window will appear with more specific details about that event.
